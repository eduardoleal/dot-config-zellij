# Zellij Configuration

Zellij terminal multiplexer config for macOS. Theme: **Catppuccin Mocha**. Shell: **fish**.

## File Structure

```
~/.config/zellij/                tracked in eduardoleal/dot-config-zellij
├── config.kdl                   keybinds, plugins, options
├── layouts/
│   └── default.kdl              zjstatus status bar, tab template
├── plugins/                     all .wasm plugin binaries live here
│   └── *.wasm
└── .claude/
    ├── settings.json            project-level hooks (committed)
    ├── settings.local.json      local permissions (gitignored)
    └── update-plugin-docs.py    hook script — regenerates plugin table below
```

This directory is a git repo. Edit `config.kdl` directly here — it's the source of truth.

## Keybinding Conventions

| Prefix | Mode/action |
|--------|-------------|
| `Ctrl p` | Pane mode |
| `Ctrl t` | Tab mode |
| `Ctrl n` | Resize mode |
| `Ctrl h` | Move mode |
| `Ctrl o` | Session mode |
| `Ctrl s` | Scroll mode |
| `Ctrl b` | Tmux mode |
| `Ctrl g` | Toggle locked |
| `Ctrl q` | Quit |
| `Ctrl e` | harpoon (pane bookmarks) |
| `Ctrl u` | room (tab switcher) |
| `Ctrl y` | zellij-forgot (keybinding cheatsheet) |
| `Alt z`  | Manual lock/unlock override (disables autolock) |

## How to Add a Plugin

1. Download the `.wasm` to `~/.config/zellij/plugins/`
2. Add a `plugins {}` alias **or** a `load_plugins {}` entry **or** a keybinding in `config.kdl`
3. Update the `KNOWN_PLUGINS` registry in `.claude/update-plugin-docs.py` with name, purpose, source, version
4. The plugin table above will regenerate automatically on the next Bash tool run

## Notes

- **autolock triggers**: `nvim|vim|hx|git|fzf|zoxide|atuin` — edit in the `plugins {}` alias in `config.kdl`
- **zjstatus colors**: Catppuccin Mocha palette hardcoded in `layouts/default.kdl`
- **Zellij version**: 0.44.1 — plugins must be built against `zellij-tile = "0.43"` or `"0.44"`; `"0.42"` is incompatible
- **Permissions**: stored in `~/Library/Caches/org.Zellij-Contributors.Zellij/permissions.kdl`; clear and restart if a plugin never prompts
